Pitwall.me

A customizable, real-time Formula 1 timing and telemetry dashboard.

Pitwall.me is a customizable, real-time Formula 1 timing and telemetry dashboard designed to provide the kind of compact, data-rich dashboard you'd find on a race engineer's console. This project is the frontend application, built with Vue 3 and Vite.

Go to Pitwall.me App View on GitHub

Features

  • Real-time Data: Connects to a data relay stream for live F1 timing and telemetry.
  • Extensive Widget Library: A wide variety of widgets for telemetry, timing, strategy, and more.
  • Customizable Layouts: Drag, drop, and resize widgets to create your perfect dashboard.
  • Multi-Page Support: Organize widgets across multiple pages.
  • Replay System: Replay timing data from past F1 sessions.
  • Import/Export: Save and share your custom layouts.

Technology Stack

The frontend is built with Vue 3, Vite, Pinia, and PrimeVue. The backend data provider is a separate project: kikkia/pitwall-drs.
